Volume I

Homelab

Infrastructure Experiments & Self-Hosted Services

Docker Linux Networking Self-Hosted
I

Overview

This volume documents the construction and maintenance of a personal homelab environment. What began as a simple curiosity evolved into a comprehensive infrastructure serving various self-hosted services and providing a sandbox for experimentation with enterprise technologies.

The homelab serves as both a practical utility and an educational endeavor, allowing hands-on experience with systems that would otherwise require expensive enterprise environments to explore.

II

Architecture

The current infrastructure consists of:

  • Proxmox VE hypervisor for virtualization
  • Docker Swarm / Kubernetes for container orchestration
  • TrueNAS for network-attached storage
  • pfSense / OPNsense for network security
  • Various self-hosted applications and services
III

Technology Stack

Virtualization

Proxmox VE, QEMU/KVM

Containers

Docker, Kubernetes

Storage

TrueNAS, ZFS

Networking

VLANs, Reverse Proxy

Monitoring

Grafana, Prometheus

Automation

Ansible, Terraform

IV

Hosted Services

  • Media server (Plex/Jellyfin)
  • Password manager (Vaultwarden)
  • File synchronization (Nextcloud)
  • Home automation (Home Assistant)
  • Network-wide ad blocking (Pi-hole/AdGuard)
  • Git repository hosting (Gitea)
  • Documentation wiki
